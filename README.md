# 🏠 Roommate Sync AI

**Smart Compatibility Matching Before Finding a Place**

Roommate Sync AI is a lifestyle-based roommate compatibility platform that helps users **find the right roommate first**, before searching for a PG or apartment together.

Instead of random co-living, the platform analyzes **daily habits, preferences, and expectations** to reduce future roommate conflicts.

The system generates **compatibility insights using AI** to help users choose roommates with similar lifestyles.

---

# 🚀 Problem Statement

Most roommate conflicts happen due to mismatched lifestyles:

• Different sleep schedules
• Cleanliness expectations
• Food preferences
• Budget comfort levels
• Work routines

Existing platforms focus mainly on **property listings**, not **human compatibility**.

This results in frequent conflicts, stress, and roommate changes.

---

# 💡 Our Solution

Roommate Sync AI flips the process:

**Step 1 — Find compatible people**
**Step 2 — Then find a place together**

The platform analyzes lifestyle data and generates:

✔ Compatibility scores
✔ Lifestyle match insights
✔ AI-generated compatibility explanations
✔ AI suggested conversation starters

This reduces friction and helps users create **better co-living environments**.

---

# 🧩 Core Features (MVP)

### 📝 Lifestyle Quiz

Users answer a short lifestyle questionnaire including:

• Sleep schedule
• Food preferences
• Cleanliness habits
• Budget expectations

---

### 📊 Compatibility Score Engine

The system calculates compatibility between users based on lifestyle alignment.

Features include:

• Dynamic compatibility scoring
• Animated circular score visualization
• Match quality indicators

---

### 📉 Compatibility Radar Chart

A visual radar chart shows compatibility across:

• Sleep habits
• Food habits
• Cleanliness
• Budget

This helps users **understand compatibility visually**.

---

### 🤖 AI Compatibility Insight

Using **Google Gemini AI**, the platform generates:

• AI explanations of roommate compatibility
• Human-readable lifestyle analysis
• Insights into potential conflicts or harmony

---

### 💬 AI Suggested Icebreaker Message

Users can generate an **AI-powered introduction message** to start conversations with potential roommates.

This helps break the ice and improve communication.

---

### 🎨 Premium Responsive UI

Modern UI built with:

• Glassmorphism design
• Animated gradients
• Responsive layouts
• Mobile-friendly interface
